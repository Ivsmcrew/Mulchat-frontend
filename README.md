# Mulchat | frontend

## About
Chat app with web + ios mobile versions based on C#-backend

## Features
- Authorization and registration
- Chatting in private / group chats
- Setting yout profile
- Web and IOS versions

## Getting started

To get the frontend running locally:

Clone this repo

`npm install` to install all the dependencies defined in a `package.json` file.

`npm run dev` to start Vite dev server.
